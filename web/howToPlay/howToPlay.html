В данном поле будут содержаться цицаты – одна или несколько. Цитаты это ключ к прохождению уровня. Возможно, указанием к прохождению уровня будет слово из цитаты. Возможно что-то другое. В общем, дерзайте и удачи вам!

— Автор проекта

В данном поле будет содержаться необходимый контент для прохождения уровня. Картинки, тексты, загадочные цифры – всё это здесь. Нужно только правильно этим воспользоваться.

В данном поле будет содержаться поле для ввода ответа. Не каждый уровень требует ответа. Существуют уровни, где это поле вообще не используется.
